Au cours de cette formation nous allons avoir 2 matières :

L'intégration

Le HTML :

Le HTML sert à créer le contenue d'un site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, etc.

Le développement :

Le Javascript :

Javascript est un langage de programmation de script employé pour rentre une page interactive, le comportement de la page et son apparence pourra varier en fonctions de certains paramètres.

Le PHP :

Le PHP est un language de programmation libre, utilisé pour générer des pages web dynamiques, qui sont donc modifiés en fonction de certaines informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisation, etc.).

MySQL :

MySQL est un système de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de création de compte) pour ensuite être ré-utilisé au moment approprié (ex : connexion à un site internet avec identifiant et mot de passe).
